Для настройки работы данного приложения необходимо:

1. Добавить вэб-сервер Tomcat 9

2. В классе com/FirstServlet.java:30 изменить путь, где будут сохраняться закачанные файлы.
    После выполния кода, файлы не удаляются.

3. В классе com/FirstServlet.java:82 изменить путь, где будут сохраняться сформированные файлы для отправки клиенту.
    После выполния кода, файлы не удаляются

4. Необходимо создать БД stoloto. Скрипт в файле resources/createDB.sql

5. В файле resources/hibernate.cfg.xml изменить настройки для подключения к БД

6. Запустить файл utils/Mail. Создадутся в БД таблицы.
